This folder is the runtime output directory for evaluation runs.

Guidance:

- Runtime evaluation outputs (e.g. `evaluation_results_*.jsonl`) are generated by the evaluation harness and are gitignored by default. Do not commit live run outputs.
- Use the `evals/WALMART_SAMPLE_QUESTIONS.json` file to run a quick sample evaluation without generating test cases.
- Keep the `.gitkeep` file so the folder remains present in the repository.

Files:

- `.gitkeep` — keep the empty folder in git.
